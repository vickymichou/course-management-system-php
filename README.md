# 🎓 Course Management System (Full-Stack)

A full-stack web application built with PHP and MySQL for managing educational content, including announcements, documents, homework, and user interactions.

---

## 📌 Overview

This project simulates a course management platform where tutors and students can interact through a structured system. It includes authentication, database-driven content, and file management functionalities.

---

## 🚀 Features

- 🔐 User authentication (login/logout with session management)
- 👥 Role-based access (students & tutors)
- 📝 CRUD operations:
  - Announcements
  - Documents
  - Homework
  - Users
- 📂 File upload and management system
- 📧 Email communication using PHPMailer
- 🗄 Relational database design (MySQL)

---

## 🛠 Tech Stack

- **Backend:** PHP  
- **Database:** MySQL  
- **Frontend:** HTML, CSS  
- **Libraries:** PHPMailer  

---

## 📂 Project Structure
-PHP files (backend logic)
-SQL file (database schema)

---

## 🧠 Key Concepts Demonstrated

- Server-side development with PHP  
- Database connectivity and query execution  
- CRUD operations and data management  
- Session handling and authentication  
- File handling and upload systems  
- Basic system design for web applications  

---

## ⚠️ Deployment Note

This project was originally deployed and tested on a university-hosted server environment.

The repository contains the full source code. Full functionality (e.g. login, database content) requires a PHP server and a MySQL database with the appropriate data.

---

## ⚙️ How to Run (Locally)

1. Install a local PHP server (e.g. XAMPP)
2. Import the provided SQL file into MySQL
3. Update database credentials in `functions.php`
4. Run the project via `localhost`
